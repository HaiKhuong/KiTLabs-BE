# Dragon mascot poses

Drop transparent PNG mascot poses here, named `<pose>.png`, matching the
`dragonPose` values used in scene specs, e.g.:

- `idle.png`
- `question.png`
- `point_left.png`
- `point_right.png`

Recommended: square-ish transparent PNGs (~1000x1000). If a referenced pose file
is missing, the renderer falls back to a transparent placeholder so the render
still succeeds.
